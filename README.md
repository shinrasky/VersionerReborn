# VersionerReborn
This is a 1.21.1 NeoForge mod.<br>
A highly flexible version checking tool, designed specifically for modpack developers.<br>
It can fetch JSON files containing version information from a fixed network location and notify players when there are updates.<br>
It also displays the current and latest version information on the main interface, as well as whether an update is needed.<br>
In addition, it can dynamically retrieve a list of sponsors from the network, JSON-formatted welcome messages, and other serializable variables.<br>
Created by shinrasky, the Versioner by TCreopargh was migrated from 1.12.2 to 1.21.1<br>
